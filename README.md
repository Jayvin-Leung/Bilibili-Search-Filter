## 注意事项

- 该项目是一款油猴脚本插件，需要依赖脚本管理器来运行，如Tampermonkey（篡改猴）或Violentmonkey（暴力猴）等
- 该脚本只在B站搜索页（search.bilibili.com）的【视频】选项下生效
- 使用该脚本时，搜索结果中的视频卡片将暂不支持视频实时预览

## 主要功能

恢复B站搜索页视频分区筛选功能
